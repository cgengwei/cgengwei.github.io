不要完美主义

从"放养"的状态一下子想转变为"极度自律",这么大的转变幅度，我敢说几乎没人能做到。

许多人因为一两次的拖延、偷懒，就开始自我否定，觉得自己太差劲、烂泥扶不上墙。

当这种负面情绪越积越多，就会给大脑施加压力，最后发展为破罐子破摔。

  • 例子1:
    当你因为偷懒贪玩而耽误了正事，不要去否定、苛责自己，从而放弃自律。最好的办法，就是利用好余下的时间，能做多少做多少。

陷入道德许可效应

心理学家调查了那些纵容自己的人，他们认为自己一直很自控，没有罪恶感，认为自己理应得到奖励，正是这种补偿的心理使人堕落！

一旦陷入了道德许可效应，你会越来越纵容自己。

该如何避免呢？

答案是，不要去做与目标相悖的事。

  • 例子1:
    有些人在学习一小时后，觉得自己做了一件光荣的事，应该得到补偿。所以奖励自己刷半小时抖音，玩几把王者。
    本来只是玩半小时，结果却陷入其中无法自拔。
  • 例子2：
    去了两天健身房，非常有成就感，这个时候，吃一顿烧烤竟然毫无心理负担。

1、执行力很重要

你是你的主人，执行力意味着你能把自己活成自己想要的模样。
时间是你宝贵的财富，执行力是你兑现财富的必要技能和思维。
  • 我曾经的领导曾跟我说，职场中执行力很重要。
  • 生活中、执行力也非常重要
  • 不要拖延、尽管去做/遇到问题就去解决
  • 可以用一些小技巧
    • 远离干扰你的手机
    • 倒计时5秒开始行动
    • 请务必保证你已经休息好了

2、睡眠决定生命的质量

给睡眠安排足够的时间，他决定你第二天的状态
  • 熬夜一时爽，第二天火葬场
  • 规律的作息，应该也是人生一堂课吧

3、如果有很多问题要解决，解决最主要的

解决主要矛盾——这应该是教员矛盾论的东西
  • 举例说明
    • 当你在学一个东西的时候，你会发现你会遇到很多东西，例如学spring源码，你会发现你需要了解gradle、了解很多很多。。。
    • 这个时候，你应当适当了解那些事情，然后快速转回来学习spring、不要因为其他事情而一直耽误这个最重要的事情。
    • 当时把spring源码看完的时候，再回首看看这些事情。看那些可以作为主要矛盾入手解决。
  • 完美主义不可取

4、当你放纵沦陷，你明知怎么做，但是你没有

做积极的选择
放纵、犯错、沉迷在所难免，愿你习得调整的方法

远离干扰源

  • 这应当成为最重要的事情，如果你在第一时间结束这种状态，应该第一时间阶段与干扰源的联系和距离

消极应对

  • 很多时候，你的大脑会一直一直提醒你，而当你不做响应
    • 你不会成为你想要成为的那个人
    • 你会获得短暂的快乐，却也获得更长久的不快乐
    • 你的大脑会很累，总是提醒一个没有完成的事情
    • 你会变得不自信，你觉得你做不到

积极应对

  • 你会成为你想要成为的那个人
  • 你的进度会一直往前走
  • 你的快乐会持续

5、学会与自己的情绪相处

当情绪不好的时候，优先调整情绪，不做任何决策和事情

6、面对失败/挫折

  • 你要明白这正是你做一件事情意义最大的事情，他能使你进步，几乎是唯一的意义
  • 不要逃避它，去面对他
  • 当发现解决了这个问题，你已经拥有针对这类问题的一个实践过的解决方案
  • 当发现问题，要解决问题，有一套非常必要的流程需要走(出自《原则》)
    步骤：
1．有明确的目标。
2．找到阻碍你实现这些目标的问题，并且不容忍问题。
3．准确诊断问题，找到问题的根源。
4．规划可以解决问题的方案。
5．做一切必要的事来践行这些方案，实现成果。

 出现问题怎么办？
1. 先把问题是什么弄明白，再决定怎么做。
2. 区分直接原因和根本原因
3. 认识到了解人（包括你自己）的特性，有助于对其形成合理预期

7、学会番茄时钟法+gtd+项目管理

番茄时钟法

  • 能让你一天都保持饱满的精神状态
  • 能让你高效地做事情
  • 能让你清晰地了解你花时间的结构

gtd

  • 能让你的脑子卸空，不再因为惦记事情而忧虑
  • 能让你清晰知道今天该做什么

项目管理

  • 能让你清晰地知道你在做什么事情、事情还有哪些步骤、进展到什么地步了

8、你要记录、你要知道你做了什么事情

记录

  • 能让你明白你做了什么事情
  • 能让你在失落难过的时候，明白这一路走过来，你已经成长了这么多（自信日记）

9、生活是个多线程

  • 你必须明白生活中总是多个事情一起进展
  • 你必须明白，你生活中可以有侧重点，但不应该忽视方方面面
  • 你必须明白，你做事的时候需要专注，但是不应该把所有事情都花在一件事

10、如果你不做计划，不做规划，那么极大可能这事永远就完成不了

  • 做计划要考虑大事要拆分成一件件小事做
  • 做计划的时候切记每天要留出时间来休息放松
  • 做计划的时候，留出可调整的空间
  • 计划做完一定要让自己能够想象按步骤做完的情况

11、生活是需要娱乐和放松的

  • 你必须将娱乐、兴趣安排到你的生活中，用心投入
